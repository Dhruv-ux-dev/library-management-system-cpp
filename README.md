# Library Management System (C++)

A console-based Library Management System implemented using **C++ and Object-Oriented Programming**.  
This project allows users to manage books in a library including adding, searching, issuing, and returning books.

---

## Features

- Add new books
- Display all books
- Search for a book by ID
- Delete books
- Issue books
- Return books
- File handling for persistent storage

---

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- File Handling
- Standard Template Library (STL)
